使用wsl2在Windows运行Linux系统

2020-02-17

  1. *”wsl” means “Windows Subsystem for Linux”.*

    wsl2 is the 2th generation of wsl , which has a much better performance.

  2. Linux[ˈlinʊks]

    此处引用陈学源同志告诉我的一句话:*”Linux is not Unix”*

  3. 使你的Windows10操作系统成为测试版（需要科学上网）

    1. 在右下角的通知栏打开设置
    2. 点击Windows预览版体验计划并加入
    3. 点击Windows更新-检查更新

    直接去微软官网下载易升获得的Windows10 1909版本尚不支持wsl2

  4. 在开始栏（小娜/控制面板）打开“启用或关闭Windows功能” 找到“适用于Windows的Linux子系统”选项并打勾，重启电脑

  5. 打开Microsoft Store，搜索Linux；建议下载Ubuntu；

  6. 下载完成后在开始栏找到它，并运行。等待几分钟后输入用户名，注意用户名的首字母不能大写，否则会报错；按下enter键以后输入密码，此时密码不可见。

  7. 以管理员权限运行powershell，键入命令wsl -l

    1. 此时如果出现wsl是外部指令，请返回步骤2；
    2. 如果能认wsl但是不认识-l，请返回步骤1；
    3. 如果没有出现你安装的Linux发行版系统，但出现了指令提示，请返回步骤3；

  8. 然后键入命令wsl --set-version <你安装的Linux发行版> 2

  9. 接着你可以在开始菜单中找到你安装的Linux发行版，运行，它现在已经可以完成独立系统可以完成的大部分操作了。但是它没有GUI，似乎是一个缺点。不过毕竟你已经拥有一个叫做Windows10的图形化操作系统了，还要什么自行车？接下来你可qian以wan尝bu试yao键入rm -rf/*命令。祝您成为一个优秀的Linuxer!

  10. return 0;

  11. /几天后补充注释：想要用wsl运行hexo极其困难，因为wsl里面Windows和Linux的环境变量部分冲突，我尝试用无数种方法试图将Linux版node.js添加到Ubuntu的环境变量，结果均以失败告终。/
